Sketching the interface

Part 1

  • Suppose we have opened the photo sharing app
  • Draw two alternate versions of an interface for selecting the photos you want to share
    • Each person on your team should draw one
  • One moment in time- can use annotations to suggest actions and causality (see below) 
  • Suppose the users then do the authentication – like they bump the phones together, authentication success, photos transferred over. 
  • Draw two alternate versions of an interface for accepting the photos on the other user’s phone
    • Each person on your team should draw one

Part 2

  • Suppose you are making an interface to a microwave
  • It should allow the user to input a time
  • It should have some way to access commonly used settings
  • It should have some indication of how much time is left
  • It should indicate if the microwave is running or not
  • It should indicate whether there is food in the microwave someone has left in and not removed. 

 

  • Draw two alternate versions of this interface 
    • Each person on your team should draw one
  • One moment in time- can use annotations to suggest actions and causality (see below) 

Remember- this may be the first time you have done this, and it is ok for it to be hard and the sketches to feel a bit rough.  PRACTICE!!
